Example git

  • 1. Tạo một repo trên github
  • 2. git init: khởi tạo
  • 3. git add: thêm một file vào git vd: git add index.html
  • 4. git commit: thêm rồi thì phải commit để đẩy nó lên mạng. VD: git commit -m "add this file to github" -m "" là thêm một message để cho người khác biết mình commit cái gì Đó là việc mình đưa file từ working vào staging (xem sơ đồ git structure) và mình commit nó đã vào local rồi
  • 5. git status: xem trạng thái của file đã được commit và có thể thêm lên mạng
  • 6. git log: kiểm tra xem file của mình đã được thêm Nhưng trên mạng chưa có.
  • 7. Đồng bộ trên mạng với máy của mình, sử dụng git remote add origin https://github.com/Lecaotanloc/git_example.git => nói cho máy của mình biết nếu máy của mình push lên thì push vào cái địa chỉ này
